.. _mongoc_server_description_t:

===========================
mongoc_server_description_t
===========================

Server description

Synopsis
--------

.. code-block:: c

  #include <mongoc/mongoc.h>
  typedef struct _mongoc_server_description_t mongoc_server_description_t

``mongoc_server_description_t`` holds information about a mongod or mongos the driver is connected to.

Lifecycle
---------

Clean up a ``mongoc_server_description_t`` with :ref:`mongoc_server_description_destroy` when necessary.

Applications receive a temporary reference to a ``mongoc_server_description_t`` as a parameter to an
SDAM Monitoring callback that must not be destroyed. See :ref:`Introduction to Application Performance Monitoring
<mongoc_application_performance_monitoring>`.
