=========
libmongoc
=========

A Cross Platform MongoDB Client Library for C

Introduction
------------

The MongoDB C Driver, also known as "libmongoc", is a library for using MongoDB from C applications,
and for writing MongoDB drivers in higher-level languages.

It depends on :ref:`libbson` to generate and parse BSON documents, the native data format of MongoDB.
